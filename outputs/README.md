# Analytical Outputs

This directory is reserved for non-source analytical artifacts created by the repository pipelines, including:
- county summary tables,
- JSON metadata files,
- report-ready Markdown tables,
- scenario outputs,
- hazard diagnostics,
- backtest tables,
- resilience scorecards.

The directory is excluded from Git tracking because these artifacts depend on the assembled acquisition layer and are regenerated rather than versioned.
